# Guia de Acessibilidade e Usabilidade: Design (UI/UX)

Este guia consolida diretrizes essenciais de Design (UI/UX) para assegurar que as interfaces sejam não apenas esteticamente agradáveis, mas funcionalmente inclusivas e eficientes. Baseado nas Heurísticas de Nielsen, na WCAG 2.2 e na nova ABNT NBR 17225:2025, este instrumento serve como guia de validação para definição visual, fluxos de navegação e prevenção de erros, garantindo um produto digital acessível para todos os perfis de usuários.
